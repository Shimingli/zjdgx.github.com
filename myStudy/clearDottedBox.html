去掉链接虚线框

在 Firefox 里可以用 -moz-outline:none; 或者 outline:none; 来将其去掉。所以我们可以这样写：

a:focus {
	 outline:none;
	 -moz-outline:none;
   }

如果想去掉输入框在chrome和safari下的阴影效果也可以用 outline 属性。

input,textarea {
	outline:none;
   }

IE。首先比较遗憾，还没有找到通过 CSS 去掉 IE 链接虚线框比较好的解决办法。所以只能用一个替代的办法，就是 a 标签的 hidefocus 属性(这个属性是 IE 独有的)。
注意：JS 脚本对应的属性名是：hideFocus。对应的 JS 代码应该是：
xxx.hideFocus ="true";
还有一种不推荐的办法。就是让链接得到焦点的时候失去焦点。
