javascript事件模型

  1. DOM0级事件处理程序:

    (1). 添加的事件在事件冒泡阶段处理;
    (2). 添加element.on+eventType = function()....
    (3). 添加的事件处理程序在元素的作用域中运行(this === element)

  2. DOM2级事件处理程序:

    (1). addEventListener, removeEventListener
    (2). element.addEventListener(type,function()...,false)
    (3). 参数: (需要处理的事件名, 事件处理函数, 调用处理程序的阶段(true: 捕获阶段, false: 冒泡阶段))
    (4). 主要好处: 可以同时添加多个事件处理程序(如多个onclick...), 并按添加顺序触发事件

  3. IE事件处理程序:

    (1). attachEvent, detachEvent
    (2). 添加element.attachEvent("on"+type,function()...)
    (3). attachEvent的事件处理程序在全局作用域中运行(this === window)
    (4). 可以同时添加多个事件处理程序, 按添加顺序相反的顺序执行

跨浏览器的事件处理程序

var EventUtil = {
	addHandler: function( element, type, handler ) {
		if ( element.addEventListener ) {
			// this === element
			element.addEventListener(type, handler, false);//冒泡阶段处理
		} else if ( element.attachEvent ) {
			// this === window
			element.attachEvent("on"+type, handler);
		} else {
			element["on"+type] = handler;
		}
	},
	removeHandler: function( element, type, handler ) {
		if ( element.removeEventListener ) {
			// this === element
			element.removeEventListener(type, handler, false);//冒泡阶段处理
		} else if ( element.detachEvent ) {
			// this === window
			element.detachEvent("on"+type, handler);
		} else {
			element["on"+type] = null;
		}
	}
}
