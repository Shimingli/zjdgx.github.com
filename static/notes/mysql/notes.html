MySQL开发Note

  • (2016/07/22) MySQL修改外键

    ALTER TABLE TableName DROP FOREIGN KEY key_name;
ALTER TABLE TableName ADD CONSTRAINT key_name FOREIGN KEY (`column_name`) REFERENCES parentTableName(`column_name`) on delete cascade on update cascade;

    (2016/07/18) MySQL增加外键

    set foreign_key_checks = 0;
alter table tableName add constraint key_name foreign key(id) references parentTableName(id) on delete cascade on update cascade;
set foreign_key_checks = 1;
// 不设置foreign_key_checks会报错

    (2013/09/22) MySQL CDATA

    CDATA标签中字符文本是不作为html被再次转义的。比如'& nbsp;'将不像其他地方那样被转换成空格。通常CDATA里面放一些‘一字不改的数据’，比如源代码，比如Javascript源码。
