移动开发中事件触发顺序

代码如下:

var box = document.getElementById('zjdgx');

box.addEventListener('touchstart', function (e) {
    console.log('touchstart: ' + e.timeStamp);
}, false);
box.addEventListener('touchmove', function (e) {
    console.log('touchmove: ' + e.timeStamp);
}, false);
box.addEventListener('touchend', function (e) {
    console.log('touchend: ' + e.timeStamp);
}, false);
box.addEventListener('mouseover', function (e) {
    console.log('mouseover: ' + e.timeStamp);
}, false);
box.addEventListener('mouseenter', function (e) {
    console.log('mouseenter: ' + e.timeStamp);
}, false);
box.addEventListener('mousedown', function (e) {
    console.log('mousedown: ' + e.timeStamp);
}, false);
box.addEventListener('click', function (e) {
    console.log('click: ' + e.timeStamp);
}, false);

结果如下:

  • touchstart: 1440470549836
  • touchend: 1440470549916
  • mouseover: 1440470550136
  • mouseenter: 1440470550137
  • mousedown: 1440470550137
  • click: 1440470550137

结论如下:

  tap事件比click事件快200多毫秒

Notes:
  1. 在mobile中, 单击才能触发mouseover和mouseenter事件, 但不能触发touchmove事件;
  2. 在mobile中, 按钮单击并拖动才能触发touchmove事件, 此时可以触发touch的三个事件;
