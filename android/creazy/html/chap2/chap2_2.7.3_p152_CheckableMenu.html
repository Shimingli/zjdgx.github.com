2.7.3 创建复选菜单项和单选菜单项(P152)代码

创建复选或单选菜单项可以调用的MenuItem的方法:

  1. setCheckable(boolean checkable): 设置该菜单项是否可以被勾选--->菜单项默认为多选菜单项;
  2. setGroupCheckable(int group, boolean checkable, boolean exclusive): 设置group组里的所有菜单项是否可勾选, 如果将exclusive设为true--->单选菜单项;
  3. setAlphabeticShortCut(char alphaChar): 设置字母快捷键;
  4. setNumbericShortCut(char numbericChar): 设置数字快捷键;
  5. setShortCut(char numbericChar, char alphaChar): 同时设置两种快捷键.

/* 复选菜单项*/
items[0] = colorMenu.add(0, RED, 0, "红色").setCheckable(true);
items[1] = colorMenu.add(0, BLUE, 0, "蓝色").setCheckable(true);
items[2] = colorMenu.add(0, GREEN, 0, "绿色").setCheckable(true);
// 设置快捷键
items[2].setAlphabeticShortcut('u');
