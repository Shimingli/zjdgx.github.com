2.7.1 菜单选项和子菜单(P146)代码

SubMenu、ContextMenu.

添加菜单(项)的方法:

  1. MenuItem add(int titleRes): 添加一个新的菜单项;
  2. MenuItem add(CharSequence title): 添加一个新的菜单项;
  3. MenuItem add(int groupId, int itemId, int order, CharSequence title): 添加一个新的处于groupId组的菜单项;
  4. MenuItem add(int groupId, int itemId, int order, int titleRes): 添加一个新的处于groupId组的菜单项;
  5. SubMenu addSubMenu(int titleRes): 添加一个新的子菜单项;
  6. SubMenu addSubMenu(CharSequence title): 添加一个新的子菜单项;
  7. SubMenu addSubMenu(int groupId, int itemId, int order, CharSequence title): 添加一个新的处于groupId组的子菜单项;
  8. SubMenu addSubMenu(int groupId, int itemId, int order, int titleRes): 添加一个新的处于groupId组的子菜单项.

SubMenu常用方法:

  1. SubMenu setHeaderIcon(Drawable icon | int iconRes): 设置菜单头的图标;
  2. SubMenu setHeaderTitle(CharSequence title | int titleRes): 设置菜单头的标题;
  3. SubMenu setHeaderView(View view): 使用View来设置菜单头.

添加菜单或子菜单步骤:

  1. 重写Activity的onCreateOptionsMenu(Menu menu)方法, 在该方法里调用Menu对象的方法来添加菜单项或子菜单;
  2. 相应菜单项点击事件: 重写Activity的onOptionsItemSelected(MenuItem mi).
